## 1. Введение
Цель моей работы - написать программу, которая поможет фиксировать нарушения в школе или универе. Я решил писать на C# потому что мы его проходили в прошлом семестре. А для кнопок и окошек взял Avalonia UI, потому что WPF старый, а мне сказали что Авалония это круто.

## 2. Архитектура и как оно работает
Программа работает с базой данных Postgres. Я там создал таблички:
- `Users` - пользователи (кто заходит в систему)
- `Reports` - жалобы (кто на кого жалуется)

Для связи с базой я использовал Entity Framework Core. Оно само делает SQL запросы, мне не надо писать INSERT INTO вручную, это удобно.
Паттерн я использовал MVVM (Model-View-ViewModel). У меня есть папка Views где лежат `.axaml` файлы, это как html только для C#. И есть папка ViewModels где логика. Чтобы всё работало, я использую RelayCommand.

## 3. Описание окон (Views)
1. **LoginWindow.axaml** - окно где юзер вводит логин и пароль. Я там сделал TextBox для пароля со звездочками.
2. **RegisterWindow.axaml** - окно регистрации. 
3. **DashboardPage.axaml** - главная страница, там просто текст и сколько жалоб всего. 

## 4. Кусок кода
Вот так у меня выглядит авторизация:
```csharp
private void ExecuteLogin()
{
    // Подключаемся к базе
    using (var db = new ApplicationDbContext())
    {
        // Ищем юзера по логину
        var user = db.Users.FirstOrDefault(u => u.Login == Login);

        // Если юзер есть и пароль совпал (проверяем хэш)
        if (user != null && BC.Verify(Password, user.PasswordHash))
        {
            // Успешный вход!
            LoginSucceeded?.Invoke(user);
        }
    }
}
```

Чтобы пароли не лежали в базе в открытом виде, я добавил библиотеку BCrypt, которая делает хэширование.
